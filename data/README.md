# 📂 Data Directory

본 프로젝트에서 사용된 데이터는 **서울 열린데이터 광장**에서 제공하는 공공데이터를 활용하였으며, 저작권 및 용량 문제로 원본 파일은 포함하지 않았습니다.

### 📊 데이터셋 정보 (Dataset Information)
- **이용객 데이터**:
    - `서울교통공사_역별 일별 시간대별 노인 승하차인원 정보.csv` (2024년 1월~2024년 12월)
    - `서울교통공사_서울시 지하철 호선별 역별 시간대별 승하차 인원 정보.csv`(2024년 1월~2024년 12월)
- **역사 편의시설 데이터**:
    - `서울시 지하철 역사 노약자 장애인 편의시설 현황.csv`(2024년 1월~2024년 12월)

### ⚙️ 데이터 구성 가이드 (Setup Guide)
분석을 재현하려면 아래의 단계를 따라주세요.
1. [서울 열린데이터 광장](https://data.seoul.go.kr/)에 접속하여 위 명칭의 데이터셋을 검색합니다.
2. 데이터를 다운로드하여 파일명을 위와 같이 수정한 후, 이 `data/` 폴더 내에 배치합니다.
3. `scripts/` 폴더 내의 전처리 및 분석 코드를 실행합니다.
